# LSTM Projects

## Overview

This repository contains a collection of projects exploring Long Short-Term Memory (LSTM) networks for sequence modeling tasks. The focus is on applying LSTM to natural language processing and time-series data to understand how sequential patterns can be learned and utilized.

## Projects

### 1. LSTM Text Generator

* Generates text sequences using LSTM
* Trained on sample text data
* Demonstrates how models learn word patterns and context

---

### 2. LSTM Sentiment Analysis

* Classifies text as positive or negative
* Uses embedding and LSTM layers for sequence modeling
* Includes preprocessing, training, evaluation, and prediction pipeline
* Evaluated using accuracy and confusion matrix

---

### 3. LSTM Stock Price Prediction

* Predicts future stock prices using historical data
* Uses time-series preprocessing and sliding window technique
* Demonstrates sequential learning for numerical data

---

## Tech Stack

* Python
* TensorFlow / Keras
* NumPy
* Pandas
* Matplotlib

## Notes

These projects are built for learning and exploration purposes. The emphasis is on understanding how LSTM networks work with different types of sequential data rather than building production-level systems.

## Future Scope

* Improving model performance with hyperparameter tuning
* Exploring deeper architectures (stacked LSTMs)
